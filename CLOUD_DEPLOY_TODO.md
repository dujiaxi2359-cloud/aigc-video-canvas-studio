# 云端部署预留清单

当前阶段不做正式云端部署，只保证公司内网共享可用。后续如果要给更多同事长期使用，可以按下面清单推进。

## 基础设施

1. 云服务器
2. 域名
3. Nginx / 宝塔反向代理
4. HTTPS 证书
5. Docker / Docker Compose 部署
6. 独立生产环境变量

## 数据与文件

1. SQLite 迁移到 MySQL / PostgreSQL，或保留 SQLite 但挂载持久化磁盘
2. 素材文件迁移到 OSS / S3 / MinIO
3. 生成结果归档策略
4. 定期备份数据库和素材

## 账号与权限

1. 用户登录
2. admin / member 权限
3. 项目 ownerUserId
4. 项目 sharedWithUserIds
5. 素材 ownerUserId
6. 团队空间

## 安全

1. API Key 只保存在后端
2. 前端不展示密钥
3. 模型配置操作日志
4. 下载鉴权
5. 上传文件类型和大小限制
6. HTTPS 强制访问

## 协作与计费

1. 模型调用额度统计
2. 每用户 / 每项目生成次数统计
3. 操作日志
4. 项目协作编辑
5. 任务队列和失败重试
6. 生成任务状态持久化

## 运维

1. 健康检查
2. 日志聚合
3. 进程守护
4. 自动重启
5. 错误报警

这份清单只是后续规划，不影响当前局域网共享版本。
